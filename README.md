# ♟️ Java Chess Engine

A console-based chess engine built in Java using object-oriented principles.

## Features
- Full 8x8 board
- Individual piece classes
- Move validation
- Turn-based system
- Expandable architecture for AI

## Technologies
- Java
- OOP
- Maven (optional)

## Future Improvements
- Minimax AI
- Alpha-beta pruning
- GUI with JavaFX
- Online multiplayer
